# Joy Crochet

## الوصف
هذا المشروع عبارة عن **صفحة ثابتة (Static Page)** لمشروع **Joy Crochet**، تهدف إلى تسهيل التواصل مع العملاء.  
الصفحة تعرض روابط لحسابات المشروع على Instagram و Facebook ورقم WhatsApp للاتصال المباشر.

يمكن زيارة الصفحة هنا: [Joy Crochet Website](https://joycrochet.netlify.app/)

## التقنيات المستخدمة
<ul dir="rtl">
  <li><strong>HTML</strong>: لبناء الهيكل الأساسي للصفحة</li>
  <li><strong>CSS</strong>: لتنسيق وتصميم الصفحة وجعلها جذابة وسهلة الاستخدام</li>
  <li><strong>Netlify</strong>: لنشر الموقع واستضافة الصفحة على الإنترنت</li>
</ul>
